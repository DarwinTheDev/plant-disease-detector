## 🚀 Quick Start

### 1. Install and Run (30 seconds)
```bash
npm install
npm run dev
```
Visit: `http://localhost:3000`

### 2. Test the App (1 minute)
- Click "Detect" tab
- Choose "Take Photo" or "Upload Image"
- See disease analysis
- Check "History" tab

### 3. Deploy (5 minutes)
```bash
npm i -g vercel
vercel
```
Or push to GitHub and connect to Vercel.

## 🔗 API Integration

The app currently uses **simulated** disease detection.

### To Connect a Real API:

1. **Edit** `lib/plant-api.ts`
2. **Replace** the `analyzePlantImage` function
3. **Add** your API endpoint and key
4. **Done!** ✅

### Supported API Options:
- PlantVillage API
- Teachable Machine
- Custom TensorFlow.js
- AWS Rekognition
- Google Cloud Vision
- Any REST API
